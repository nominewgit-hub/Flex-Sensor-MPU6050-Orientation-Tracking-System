# Flex-Sensor-MPU6050-Orientation-Tracking-System
A real-time motion tracking system using Arduino with a Flex Sensor for bend angle measurement and MPU6050 IMU for 3-axis orientation (roll, pitch, yaw). Features sensor fusion via complementary filter, active gyro drift compensation, noise reduction, and live serial output. Perfect for gesture recognition, robotics, or wearable tech projects.
